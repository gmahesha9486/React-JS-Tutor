npm create vite@latest
08miniContext

consider context as global variable

Creation of Context.
Step 1: UserContext.js file
import React from 'react'
const UserContext = React.createContext()
export default UserContext;

Step 2: now, create UserContextProvider.jsx file
import React, { useState } from 'react'
import UserContext from './UserContext';

const UserContextProvider = ({children}) => {

  const [user, setUser] = useState(null);  
  return (
    <>
        <UserContext.Provider value={{user, setUser}}>
        {children}
        </UserContext.Provider>
    </>
  )
}

export default UserContextProvider


Step 3: write this in App.jsx
 return (
    <UserContextProvider>
      <h1>React tutor</h1>
      <Login />
      <Profile />
    </UserContextProvider>
  )

Note: Sometimes, team can write this in App.jsx file
<UserContext.Provider value={{user, setUser}}>

write this code in Login.jsx
const {setUser} = useContext(UserContext)
note: setUser is defined in the file: UserContextProvider.jsx
